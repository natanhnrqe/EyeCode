# Learning Content

Este diretório conterá os conteúdos Markdown do Learning.

Os arquivos serão migrados gradativamente nas próximas sprints,
substituindo o `MarkdownBuilder` como fonte de conteúdo.

Estrutura esperada:

```
learning/
  class.md
  interface.md
  enum.md
  record.md
```

Cada arquivo é um documento Markdown completo, compatível com o
`MarkdownParser` já existente no sistema.
